# 🎓 GATE AI Assistant 🤖

An AI-powered assistant designed to help students prepare for the **GATE (Graduate Aptitude Test in Engineering)** by providing accurate, contextual, and intelligent responses using advanced AI models.

---

## 📌 About the Project

The GATE AI Assistant is an intelligent system built using **Generative AI (Gemini/OpenAI APIs)** that helps students solve doubts, understand concepts, and access study material efficiently.

It integrates **Retrieval-Augmented Generation (RAG)** to deliver **accurate and context-aware responses**, making it more reliable than traditional chatbots.

---

## 🚀 Features

- 🤖 AI-powered doubt solving  
- 📄 Supports PDF, DOCX, TXT file uploads  
- 🔍 Retrieval-Augmented Generation (RAG)  
- 🧠 Context-aware intelligent responses  
- 📥 Download answers as PDF  
- 🎨 Interactive UI using Streamlit  
- ⚡ Fast and user-friendly

## 💻 Installation & Setup

### 1️⃣ Clone the Repository
git clone https://github.com/Anuj5642/Gate-AI-Assistant.git
cd Gate-AI-Assistant

### 2️⃣ Install Dependencies
pip install -r requirements.txt

### 3️⃣ Setup Environment Variables
GEMINI_API_KEY=your_api_key

### 4️⃣ Run the Application
streamlit run app.py
